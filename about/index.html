Features Overview

  • Form for Data Entry: Users can input artist information conveniently.
  • Display Functionality: View added artists in a table format.
  • Update Records: Modify existing artist details easily.
  • Deletion Feature: Remove artists from the table effortlessly.
  • Mark Artists as Featured: Segregate certain artists into a separate table.
  • Search Functionality: Find artists based on their ID quickly.

Technology Stack

  • Built with Angular and TypeScript for robust and scalable web applications.
  • Utilizes HTML and CSS for structure and styling..
  • Employs Reactive Forms module from Angular for efficient form handling.
  • Integrates Router Modules from @angular/router for seamless navigation within the application.

Instructions for Use:

  • Add Artists: Fill out the form with artist details and click the "Add Artist" button.
  • Display Artists: Click on the "Display" button to view the added artists in a table.
  • Update Records: Click on the "Update" button artist's row in the table to edit their information.
  • Delete Artists: Click on the "Delete" button next to an artist's row to remove them from the table.
  • Mark as Featured: Check the checkbox for featured artist to keep them as "featured" and in a featured table.
